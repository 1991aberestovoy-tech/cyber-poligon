# Киберполигон: CTF за 45 минут

Статическое веб-приложение для школьного CTF-турнира по кибербезопасности.

## Что внутри

- 2 CTF-трека по 10 заданий разной сложности.
- Каждый трек рассчитан на 40-50 минут.
- Категории: Web, Crypto, Forensics, OSINT, Logs, Linux, DevSecOps и Incident Response.
- Без регистрации и сервера.
- Прогресс хранится только в браузере участника.
- Есть таймер, баллы, финальный профиль и подсказки по направлениям в кибербезопасности.

## Как запустить

Откройте `index.html` в браузере.

## Как сделать QR-код

1. Загрузите папку на статический хостинг: GitHub Pages, Netlify, Vercel, школьный сайт или внутренний сервер.
2. Скопируйте публичную ссылку на `index.html`.
3. Создайте QR-код на эту ссылку любым генератором QR.
4. Перед мероприятием проверьте ссылку с телефона в той же сети, где будут школьники.

## Рекомендация для занятия

Выберите один CTF-трек и дайте участникам 45 минут на прохождение, затем 5-10 минут на обсуждение:

- какие задания были самыми сложными;
- какие роли показались интересными;
- где в реальной жизни встречаются Base64, robots.txt, логи, cookie, JWT, IDOR, SQL-инъекции, SUID и секреты в контейнерах.
